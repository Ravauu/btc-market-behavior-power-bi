# BTC Market Behavior Dashboard

Power BI project analyzing BTC 5-minute market behavior using Binance perpetual futures data.

The dashboard focuses on intraday volatility, candle structure, trading activity, taker buy share and differences between market sessions.

## Project Overview

The dataset contains approximately 139,680 five-minute BTCUSD perpetual candles from January 2025 to April 2026.

The project was created to explore how BTC behaves across different hours, days of the week and trading sessions. The analysis is focused on market behavior and descriptive analytics, not on generating trading signals.
